Paramètres

🌍 World Settings

Cette section permet de configurer l'apparence globale du monde :

  • Global Illumination : Ajustable via un slider. Plus la valeur est élevée, plus le niveau est sombre.
  • Parallaxe : Choisissez un fond animé (parallax) ou une couleur unie.

Les parallax sont définis dans Assets/backgrounds.json. Voici un exemple de configuration :

{
  "name": "Apocalyptic",
  "type": "image",
  "bg_color": [50, 40, 40],
  "layers": [
    { "path": "Assets/images/Parallax/Apocalyptic/1.png", "parallax": 0.05 },
    ...
  ]
}
💡 Ajouter votre propre parallax :
- Créez un dossier dans Assets/images/Parallax/
- Placez-y vos images
- Ajoutez une entrée dans backgrounds.json avec leurs chemins et coefficients de parallaxe
- Redémarrez l'éditeur pour les voir dans la liste 
{
  "name": "Solid Gray",
  "type": "color",
  "color": [200, 200, 200]
}
🎨 Utiliser une couleur unie comme fond :
- Définissez "type": "color"
- Renseignez la couleur RGB avec la clé "color"
- Exemple : [200, 200, 200] pour un gris clair
- Cette option permet d'avoir un fond statique sans effet de parallaxe

👤 Player Profile

Définit les caractéristiques du joueur pour le mode Play :

  • Vitesse (Speed)
  • Gravité (Gravity)
  • Force de saut (Jump Force)
  • Can Fly : Activer pour permettre au joueur de voler (utile pour le debug)
  • Spawn Point : Vous devez avoir sélectionné un Location Point puis cliquer sur Set Spawn Point. Le point affichera alors des drapeaux croisés.
Attention :
Le mode Play ne peut pas être lancé sans point de spawn défini.

⚙️ Paramètres Avancés

Cette section permet d'afficher ou masquer certains éléments :

  • Display Lights : Affiche les sources de lumière
  • Show Collisions : Affiche les zones de collision
  • Show Location Points : Affiche les points de localisation
  • Display Grid : Affiche la grille d'édition
  • Display FPS : Affiche les performances de rendu
  • Keyframe Overlay : Affiche un surlignage jaune sur les tuiles avec animation
  • Display VFX : Affiche ou masque les particules d'effets visuels dans le viewport et en jeu

🧰 Start Mode

Détermine le niveau chargé au démarrage :

  • Open NewExample : charge un nouveau niveau exemple
  • Open OldExample : charge l'ancien exemple
  • Empty Project : démarrage avec un projet vide
  • Open Last Level : recharge automatiquement le dernier niveau utilisé (⚠️ à éviter en cas de crash)
⚠️ En cas de problème :
Si l'éditeur refuse de démarrer ou plante à cause de paramètres corrompus (comme un Open Last Level sur un fichier manquant), vous pouvez :
  • Supprimer le fichier Assets/ui/settings_ui pour réinitialiser les paramètres
  • Ou bien l'ouvrir manuellement avec un éditeur de texte pour corriger les valeurs
